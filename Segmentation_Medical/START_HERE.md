# Medical Image Segmentation Project

This directory contains comprehensive implementations of medical image segmentation techniques.

## Quick Links
- **README.md**: Full project documentation
- **requirements.txt**: Python dependencies
- **notebooks/**: Jupyter notebooks with examples
- **utils/**: Data loading, metrics, visualization
- **models/**: U-Net 3D and MedSam implementations

## Getting Started
```bash
pip install -r requirements.txt
jupyter notebook
```

For detailed information, see README.md
